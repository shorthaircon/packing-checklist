# 出國行李清單

一個用瀏覽器打開就能用的打包清單小工具。開啟即用、完全離線、資料只存在自己的瀏覽器。

🔗 **立即使用**：https://shorthaircon.github.io/packing-checklist/

---

## 功能

### 核心
- **6 大預設分類** · 隨身物品、衣物、個人用品、飛機用品、Extra、記得確認
- **一秒勾選** · 點整列任何地方都能勾／取消勾，伴隨羽毛飛出動畫
- **頁首進度條** · 紙飛機沿著進度條飛，一眼看到還有多少沒打包
- **全部完成慶祝動畫** · 勾完最後一項會有小驚喜

### 最終檢查模式
- 點頁首的夕陽紅按鈕進入「最終檢查」
- 只顯示標星星（重要）的分類，出門前最後確認護照、證件、網路卡等不漏帶
- 全部重要項目勾完自動退出，提示「可以出門了 ✈」

### 自訂清單
- 新增／刪除／改名分類，從 36 個圖示中挑一個
- 新增／刪除／改名項目
- 拖曳排序（分類、項目都能拖）
- 標星星 ★ 把分類設為「重要」，會進最終檢查

### 離線可用（PWA）
- 加到手機主畫面，像 App 一樣開
- 首次載入後完全離線可用（飛機、國外無網路都 OK）
- 資料存在瀏覽器的 localStorage，不上傳任何雲端

## 安裝到手機主畫面

**iPhone Safari**
1. 打開網址
2. 按下方分享按鈕
3. 選「加入主畫面」

**Android Chrome**
1. 打開網址
2. 右上角選單
3. 選「加到主畫面」或「安裝應用程式」

---

## 重設

- **勾選重設**（頁首 🔄）：清空所有勾選，保留分類與項目。適合下次出國前。
- **完全重設**：清除瀏覽器的 localStorage，會回到預設清單。

## 設計風格

旅行手帳風 — 溫暖紙質感、手寫字體、輕量插畫元素。

- 標題：LXGW WenKai TC（手寫感）
- 內文：Noto Serif TC
- 主色：紙張米白 `#FAF6EB`、墨黑 `#2A2E33`、夕陽紅 `#C85C5C`、天空藍 `#7EC8E3`
- 裝飾：紙膠帶角、飛機、雲朵、羽毛、橡皮章

支援 `prefers-reduced-motion`：偏好減少動態的使用者不會看到特效。

## 技術

純靜態網站 — 沒有框架、沒有後端、沒有依賴任何服務。

- HTML / CSS / 原生 JavaScript（IIFE 單檔）
- [Lucide](https://lucide.dev/) icon set
- [SortableJS](https://sortablejs.github.io/Sortable/) 拖曳排序
- Service Worker + manifest.json 提供 PWA 能力
- 資料儲存：`localStorage`

所有外部資源（字體、圖示、JS 庫）皆打包在 repo 內，無 CDN 依賴。
